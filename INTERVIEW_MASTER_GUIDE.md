## üõ† 6. The "Aaron Maxwell" Advice Column
**Q: "I struggle with reading the problem."**  
**A:** Use the [Pseudocode Methodology](./PSEUDOCODE_METHODOLOGY.md). Read it three times. Identify the constraints.

**Q: "How do I sort by two things at once?"**  
**A:** Python Tuple Sorting. `data.sort(key=lambda x: (x.primary, -x.secondary))`. See [Advanced Algorithms](./ADVANCED_DEVOPS_ALGO_GUIDE.md).

**Q: "Do I need to memorize Knapsack?"**  
**A:** No, but you need to recognize it. If you see "Maximize resource usage with a limit," that is Knapsack. See "Server Bin Packing" in [Advanced Algorithms](./ADVANCED_DEVOPS_ALGO_GUIDE.md).
